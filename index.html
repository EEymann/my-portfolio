  1. Hello World.

    This is the very first webpage I learned how to make! Then, to add more information to compliment the header, we can add an h2 tag, which will be a little smaller than the h1 tag. I also learned about p tags, which allow us to write paragraphs, like this one. Finally, I learned about ordered and unordered lists, which create nice bullet points or numbers to a list of items, like this list of my projects. Pretty neat!

  7. My Dream Vacation

    This webpage is written using HTML and the power of bootsrap. Here you can learn about The Fiji Islands, a dream vacation destination of mine! Bootstrap allowed me to add in some pretty awesome features to my page, such as a container, to pad all content; a Jumbotron, to make my title stand out; blockquotes, to display quotes from previous tourists in the area; and helper classes, to highlight important information. Notice the responsive design? That's all thanks to bootstrap!
